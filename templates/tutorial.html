Rules

The investment game is a two player, turn based game

Each turn, one player sends points to the other

This action represents doing something like

  • doing somebody a favor
  • investing in a startup
  • lending money to a friend

Each round of the game has four (4) steps

Player A receives $10

Player A sends $0-10 (M) to Player B and keeps the rest ($10-M)

Player B receives 3 x M

Player B sends $0-3M back to Player A

Each game has five (5) rounds
